# Kubi German Sentences

This project provides a progressive collection of German sentences for educational purposes, structured by difficulty levels.

## Structure
- **index.html**: Main entry point for the web interface.
- **noun_verb_adjective.html**: Additional interface for practicing nouns, verbs, and adjectives.
- **sentences.js**: Contains German sentences organized by levels, from simple to complex.

## Levels
1. Simple sentences
2. Sentences with objects
3. Sentences with prepositions
4. Sentences with two adjectives
5. Longer phrases
6. Complex sentences

## Usage
Use the sentences for language learning, grammar practice, or as examples in educational apps.
